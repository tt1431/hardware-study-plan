## 本章导引

前几章我们学完传输线理论、反射与端接、串扰之后，这两章进入了信号完整性中两个承上启下的核心话题：

- **Ch7 阻抗不连续**：反射学的延伸。不只是"源端 vs 负载端"的阻抗失配，而是互连线**沿途每一处微观结构**(分支、过孔、跨分割、参考平面变形)都可能造成局部阻抗突变，引发反射。这一章的核心问题是——**哪些 PCB 结构会破坏阻抗连续性？为什么？怎么处理？**

- **Ch8 差分互连**：高速信号新范式。在现代高速总线 (PCIe、USB、LVDS、DDR 差分时钟) 中，差分信号是标配。这一章告诉我们：差分信号怎么传输？返回电流去哪？为什么差分抗干扰？——本质是**从"单线 + 地平面"思维，跃迁到"差分对 + 共模抑制"思维**。

打通这两章，你就能理解为什么 DDR 绕线要等长、为什么差分对下方一定要完整参考平面、为什么过孔是 SI 的噩梦。

## 7.1 分支结构——一根线分两叉，阻抗直接对折

### 物理本质

信号沿传输线传播，每步感受到一个瞬时阻抗 $Z_0$ 。当信号到达一个 **T 形分支点** 时，看到的是两条传输线的并联：

$$ Z_{branch} = Z_0 \parallel Z_0 = \frac{Z_0}{2} $$

**一句话**：分支点处，阻抗从 $Z_0$ 跌到一半，产生负反射。

### 关键细节

1. 这只是在理想情况下的理论值。实际中由于信号上升时间 $T_r$ 不为零，
   信号 "感受" 到的阻抗不会真的跌到 $Z_0/2$ ——
   分支越短、 $T_r$ 越大，感观阻抗越接近
   $Z_0$ 。

2. 分支越长，等效电容越大，阻抗跌落越深，反射越严重。这就是为什么 **菊花链拓扑必须尽量减少分支残桩 stub 长度**。

3. 分支结构本质上是一个 **带宽受限结构**——速率上去之后，根本不能再用分支拓扑。PCIe、USB3 这种高速信号，只允许 **点对点** 连接。

> **页码参考**：§7.1 pp.211-213(分支结构的阻抗模型)

## 7.2 参考平面高度变化——你脚下的"地"被挖了

### 场景

PCB 上参考平面并非永远完整。典型场景：

- 连接器区域：为安装连接器，其下的铜皮被掏空
- 安装孔、定位孔周围：反焊盘 (anti-pad) 掏空了参考平面
- BGA 区域：密集的过孔排阵，参考平面被啃成筛子

### 物理本质

参考平面变窄或被掏空 → 信号线和参考平面之间的距离 $h$ 等效变大 →
   单位长度电容 $C$ 减小 →
   特性阻抗 $Z_0 = \sqrt{L/C}$ 升高。

$$ Z_0 \uparrow \quad \text{when} \quad h \uparrow $$

### 关键结论

- **外延值**：掏空区域到走线中心线的距离。外延值越小，阻抗变化越大；外延值足够大时，阻抗趋于稳定回到 $Z_0$ 。
- 两侧都掏空比一侧掏空影响更大。
- 内层走线受掏空影响比表层走线**小**(因为内层上下都有参考)。

> **页码参考**：§7.2 pp.213-215(参考平面被掏空的影响)

## 7.3 互连线跨分割——回流的"路"断了

### 场景

多层板中，电源层经常被分割成多个电源域 (1.8V / 3.3V / 5V 等)。当信号线跨越不同电源区域的分割线时，参考平面被切断。

### 物理本质

返回电流必须走阻抗最小的路径——也就是信号线正下方的参考平面。跨分割让这个路径被物理截断，返回电流只能 **绕路走**：

- 绕路 → 回路面积增大 → 回路电感 $L_{loop}$ 增大
- $L_{loop}$ 增大 → 局部阻抗升高 → 反射
- 分割处的电磁场向外泄漏 → **串扰也增大**

### 三方面后果

| 后果 | 机制 | 严重程度 |
|:--|:--|:--|
| 反射 | 局部阻抗突变 | $T_r$ 越小越严重 |
| 串扰 | 返回路径绕行，耦合到相邻信号 | 高速时不可忽略 |
| EMI | 分割处电磁场散射 | 差分信号可缓解 |

### 工程守则

敏感高速信号**绝对不要跨分割**。迫不得已时，在分割线下方放一颗缝合电容 (stitching capacitor)，提供交流回流路径。

> **页码参考**：§7.3 pp.215-218(互连线跨分割)

## 7.4 过孔——SI 工程中最"坑"的结构

过孔是 PCB 上最常见的阻抗不连续点。一个典型的通孔型过孔由**焊盘 + 孔壁 + 反焊盘 + 残桩**四部分组成，每一部分都影响阻抗。

### 过孔结构剖析

```
       ┌──── 信号线 ────┐
       │                 │
  ┌────▼───┐  ← 焊盘 (pad), 容性
  │  ┌───┐ │
  │  │ ○ │ │  ← 孔壁 (barrel), 主要为感性
  │  └───┘ │
  └────────┘
       │    ← 反焊盘 (anti-pad), 空隙越大阻抗越高
  ─────┼────  参考平面
       │
       ▼    ← 残桩 (stub), 悬空的过孔尾巴
```

### 影响过孔阻抗的四个因素

| 因素 | 影响方向 | 物理原因 |
|:--|:--|:--|
| 焊盘直径 ↑ | 阻抗 ↓ | 寄生电容增大 |
| 反焊盘 ↑ | 阻抗 ↑ | 减少了对参考平面的寄生电容 |
| 非功能焊盘去除 | 阻抗 ↑ | 减少了寄生电容 |
| 残桩长度 ↑ | 阻抗 ↓ | 残桩形成容性负载 |

### 工程最佳实践

1. **高速信号**：减小焊盘尺寸，增大反焊盘
2. **内层非功能焊盘** (不与走线连接的焊盘)：直接删掉，减小寄生电容
3. **残桩**：背钻 (back-drill) 去除，这是 10Gbps+ 信号的标配操作
4. **地过孔**：高速信号换层时旁边加回流地过孔

> **页码参考**：§7.4 pp.218-222(过孔结构分析)

| 不连续来源 | 阻抗变化 | 反射方向 | 缓解方法 |
|:--|:--|:--|:--|
| 分支结构 | ↓ (最低 $Z_0/2$) | 负反射 | 减小分支长度, 用点对点拓扑 |
| 参考平面掏空 | ↑ | 正反射 | 增大外延距离 |
| 跨分割 | ↑ | 正反射 | 避免跨分割, 加缝合电容 |
| 过孔 | ↓ | 负反射 | 背钻残桩, 去非功能焊盘, 增大反焊盘 |

## 8.1 差分信号的基本原理

### 为什么需要差分

单端信号：一根信号线 + 一根地 (GND)，信号电压以地为参考。  
差分信号：
   两根线 $D_+$ 和
   $D_-$ ，
   **信号是两根线之间的电压差**。

$$ V_{diff} = V_+ - V_- $$

核心优势：
- **抗共模噪声**：外部干扰同时耦合到两根线上 → 差值不变
- **EMI 辐射低**：两根线上的电流方向相反，产生的磁场在远处相互抵消
- **摆幅可减半**：同样噪声容限下，每根线只需一半摆幅 → 功耗降低

### 共模 vs 差模

| | 差模 (Differential Mode) | 共模 (Common Mode) |
|:--|:--|:--|
| 定义 | $V_+$ 和 $V_-$ 反向跳变 | $V_+$ 和 $V_-$ 同向跳变 |
| 承载信息 | ✅ 有用的数据 | ❌ 不需要的噪声 |
| 来源 | 驱动器输出 | 外部干扰、地弹、不对称 |

## 8.2 共模抑制能力

差分接收器有一个关键指标：**共模抑制比 (CMRR, Common-Mode Rejection Ratio)**。
和 $V_-$ 同时被抬高或拉低 (共模电压变化)，
   只要差值 $V_{diff}$ 保持不变，
   接收器仍然能正确判断 0 和 1。

> **类比**：两个人在狂风中被同时吹向同一个方向——两个人之间的距离没变，所以信息没丢。

但是注意：共模抑制有范围限制。如果共模电压超出了接收器的输入共模范围 (Input Common-Mode Range)，接收器就会失效。这就是为什么我们不能让差分对的共模电压随便飘。

> **页码参考**：§8.1-8.3 pp.223-226(差分信号基础 + 共模抑制)

## 8.3 差分对的返回路径——关键不是对方，是地平面

### 常见误解

"差分信号是一对互补信号，一条线的返回电流走另一条线。"——**这话只说对了一小部分。**

### 真相

差分对中每条信号线的**返回电流主要仍然集中在参考平面上**，而不是在另一条走线上。

物理原因：
- 两根差分线间距通常远大于信号线与参考平面的距离
- 回路电感最小的路径是正下方的参考平面，不是旁边的差分线

所以：**参考平面对差分对的重要性，和单端信号一样大。** 差分对下方的参考平面断了，信号照样会出问题。

> **页码参考**：§8.4 pp.226-228(差分对的返回电流分布)

## 8.4 差分对走线——平行不是必须，对称才是关键

### 核心原则

差分互连不一定要用平行走线。只要做到 **两条线周围电磁环境一致**，信号就能正常传输。

这意味着：
- 两条线可以在不同层走，只要周围介质、参考平面一致
- BGA 扇出时，差分对经常被迫在不同层分开走一段——只要走线长度匹配、环境对称，就是可接受的

### 为什么平行更常用

- 平行走线时，两根线受到的外部干扰几乎完全相同 → 共模噪声更容易在接收端被抵消
- 平行走线便于**等长控制**
- 差分阻抗 $Z_{diff}$ 约等于
  $2 Z_{odd}$ ，更容易设计和仿真

### 差分对的关键约束

1. **等长** (Length Matching)：两根线长度必须匹配，偏差通常在 5 mil 以内
2. **等间距**：线间距保持恒定，确保差分阻抗连续
3. **完整参考平面**：下方参考平面必须连续，不能跨分割
4. **对称性**：过孔对数、线宽变化保持对称

> **页码参考**：§8.5-8.6 pp.228-230(差分互连的走线策略)

## 8.5 差分与共模的相互转化

现实中，差分信号不是完美的。以下情况会将差模能量转化为共模噪声：

| 转化来源 | 机制 |
|:--|:--|
| 两条线不等长 | 一个信号先到，另一个后到 → 时间差产生共模分量 |
| 两条线阻抗不匹配 | 反射系数不同 → 两个信号幅度不同 → 差模能量残余为共模 |
| 过孔不对称 | 两根线过孔数或类型不同 → 延迟和阻抗差异 |
| 参考平面不连续 | 线1和线2下方的参考环境不同 → 不对称耦合 |

### S 参数标记法

差分 S 参数用四个端口描述差分对的特性：

- $S_{DD21}$ ：差分输入 → 差分输出 = **差模传输**
- $S_{CC21}$ ：共模输入 → 共模输出 = 共模传输
- $S_{CD21}$ ：差分输入 → 共模输出 = **差模→共模转化** (越小越好)
- $S_{DC21}$ ：共模输入 → 差分输出 = 共模→差模转化

工程上， $S_{CD21}$ 是衡量差分对 "干净程度" 的关键指标。

> **页码参考**：§8.7-8.8 pp.230-233(差分与共模的相互转化 + 差分 S 参数)

### Ch8: 差分互连速查

| 概念 | 要点 |
|:--|:--|
| 差分信号 | $V_{diff} = V_+ - V_-$ , 抗共模干扰 |
| 共模抑制 | 接收端能忍受的共模电压范围, 超出则失效 |
| 返回路径 | 主要走参考平面, 不是对方走线 |
| 走线关键 | 对称 > 平行; 等长 + 等间距 + 连续参考平面 |
| 差模→共模转化 | 不等长/不匹配/不对称 → $S_{CD21}$ |

## 面试/工程问题

**Q1: 为什么差分信号比单端信号抗干扰能力强？**

A: 外部噪声同时耦合到
   $D_+$ 和
   $D_-$ 上，
   形成共模干扰。
   差分接收器只检测差值 $V_+ - V_-$ ，
   共模分量被抑制。
   这在物理上等价于 "两个人同时被风吹，距离不变"。

**Q2: 差分对下方参考平面断了会怎样？**

A: 差分对每条线的返回电流仍然主要走参考平面, 不是走对方。参考平面断裂会导致局部阻抗突变 → 反射, 还会破坏两条线的对称性 → 差模能量转化为共模噪声 → 信号质量恶化。

**Q3: 为什么 BGA 扇出时差分对要打 GND 过孔？**

A: 信号换层时, 返回电流也需要换层。如果不在换层过孔旁边加 GND 过孔, 返回电流没路走, 会绕行产生巨大的回路电感 → 阻抗突变 + 串扰。GND 过孔为返回电流提供低阻抗的跨层通道。

**Q4: 过孔的"残桩"为什么非去掉不可？**

A: 残桩是一段悬空的开路传输线, 其末端 $Z \to \infty$ 造成全反射。在特定频率 (残桩长度为 1/4 波长时) 会形成谐振陷波, 严重衰减信号。10Gbps+ 信号必须背钻去除残桩。

**Q5: PCB Layout 中发现高速差分对必须跨一次分割, 怎么处理？**

A: 在分割线下方、差分对正下方的两侧各放一颗 0.1µF 的缝合电容, 为返回电流提供交流跨接路径。但这是被迫方案, 最优解是重新规划层叠, 让差分对有完整参考平面。
