# 信号完整性学习笔记 ①｜什么是信号完整性？

> 📚 《信号完整性揭秘：于博士 SI 设计手记》  
> 📅 2026-06-10

---

## 1.1 数字信号 vs 模拟信号

先说一个很多人忽视的基本事实：**数字信号是用模拟波形承载的。**

在低速时代（kHz 到几十 MHz），这事不重要。信号边沿缓、带宽窄，PCB 走线近似理想导线，0 就是 0，1 就是 1，工程师只要确保"通不通"即可。

但频率上去以后，电压/电流波形在传输线中的行为完全遵从模拟电路的物理规律——反射、衰减、耦合、谐振，一个不落。0 可能被误判成 1，1 可能被误判成 0。

> 于博士的核心观点：**数字信号活在模拟世界里。** 频率一高，所有模拟效应全部暴露。

打个比方：低速时代你在小路上骑自行车，路面平整就好；高速时代你在高速上飙车，路面每一个坑洼（阻抗不连续）、护栏反射（端接不匹配）、隔壁车道的干扰（串扰）全部放大显现。

---

## 1.2 一句话概括信号完整性

> 信号完整性（Signal Integrity, SI）：由互连线、电源分配网络、器件等引起的所有信号质量偏差及延时问题。核心就是保证 0/1 码流被准确传到接收端并被准确判断。

于博士把 SI 拆成四大块：

| 领域 | 研究什么 | 典型问题 |
|:---|:---|:---|
| 反射 SI | 阻抗不连续引起信号反弹 | 过冲、下冲、振铃、台阶 |
| 串扰 SI | 相邻通道互相干扰 | 噪声耦合、边沿抖动 |
| 电源完整性 | 电源分配网络噪声 | 地弹、同步开关噪声 |
| 时序 SI | 信号到达时间不一致 | 建立/保持时间违规 |

---

## 1.3 SI 问题的根源：上升时间

这是第一章最重要的结论，值得反复吃透。

### 表面现象 vs 深层原因

很多人以为"工作频率高了 SI 问题就来了"，这没错但不准确。真正的原因藏在更底层：

```
芯片工艺进步（28nm → 14nm → 7nm）
    ↓
晶体管沟道长度缩短
    ↓
晶体管开关时间缩短
    ↓
信号上升时间 Tr 越来越小  ← 根源
    ↓
高频分量增多
    ↓
PCB 寄生参数效应暴露
    ↓
波形畸变
```

举个反直觉的例子：一个 **10MHz** 的极陡方波（Tr=0.1ns），高频分量比一个 **100MHz** 缓边沿时钟（Tr=5ns）还多。SI 设计师看的是上升时间和带宽，不是工作频率。

### 恶性循环

芯片越来越快 → Tr 越来越小 → SI 越来越难搞。但芯片必然继续变快，所以 SI 永远有活干——这是饭碗，不是 bug。

## 1.4 知识体系是网，不是链

于博士在 1.4 小节有段很关键的提醒：单独看每一章都能看懂，但综合运用是另一回事。你为优化一个问题做的操作，可能在其他地方埋了雷。

举个例子：

```
想减小串扰 → 拉开两线间距 → 走线变长 → 延时增大 → 时序出问题
```

SI 设计的三个目标天然矛盾：

- 信号质量（阻抗匹配、端接）
- 成本（少加层、少加器件）
- 面积（走线短、器件小）

好的 SI 工程师是**平衡艺术家**，不是按图施工的流水工。你优化一个，得同时盯着另外两个。

## 1.5 电平有效 vs 边沿有效

| | 电平有效 | 边沿有效 |
|:---|:---|:---|
| 触发方式 | 高/低电平期间 | 上升/下降沿瞬间 |
| 时序敏感度 | 低 | **高** |
| 典型应用 | EN、CS、RST | SPI 时钟、DDR 时钟 |

边沿有效对时序敏感的根本原因：数据只在跳变瞬间被采。时钟上升沿因串扰或反射抖动 50ps，接收端就可能采到错误数据。50ps 在低速时代不算什么，在 DDR5 时代就是致命问题。

## 1.6 本章涉及的 SI 设计规则

### 高速信号源端串联 33Ω

很多人背"高速信号串 33Ω"，但不知道为什么。

```
驱动端 ──[33Ω]──── 50Ω传输线 ──── 接收端
```

原理是源端阻抗匹配：驱动端输出阻抗通常 10~20Ω，加上 33Ω 后接近 50Ω，和传输线特性阻抗相等。信号先以半幅度传到接收端 → 接收端高阻全反射 → 反射波回来叠加到满幅度 → 回到源端时已匹配不再反射。

源端波形会出现"台阶"，台阶宽度就是信号往返时间。33Ω 不是死的——线是 65Ω 你得重新算。

### 3W 原则

相邻信号线中心间距 ≥ 3 倍线宽。

为什么是 3W？实验数据：

- 1W 间距 → 串扰 30%~40%
- 2W 间距 → 串扰 10%~15%
- **3W 间距 → 串扰 <5%** ← 拐点

3W 之后继续拉大改善有限，浪费面积。拉开间距是减小串扰最直接有效的手段。

### 20H 原则

电源平面边缘比地平面内缩 20 倍层间距（H = 电源-地介质厚度）。

PCB 板边电磁场会向外辐射 EMI。电源平面内缩 20H 后，边缘场被地平面包住，辐射下降约 70%。继续加大到 100H 也不超过 10% 改善——20H 是"够用不浪费"的拐点。

### 时钟走内层

两个原因：

1. **消除远端串扰**。表层微带线介质不均匀（上空气下板材），奇模偶模速度不同 → 远端串扰大。内层带状线上下介质均匀 → 远端串扰近似为零。
2. **屏蔽 EMI**。内层走线被上下参考平面包住，电磁场不外泄。
