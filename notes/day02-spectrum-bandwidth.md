## 2.1 一个前提：同一根线，不同频率，不同态度

做硬件的人每天都在画 PCB 走线。在低速时代，一根线就是一根导体——连上就行。

但到了高速，同一条走线对不同频率的信号分量的"脾气"完全不同。说说为什么。

### 四个物理效应

**寄生电感。** 任何导体都有自感，PCB 走线也不例外。感抗的公式是：

```
X_L = 2πfL
```

频率 f 翻 10 倍，感抗翻 10 倍。10MHz 下走线电感几乎不构成阻碍，到了 1GHz 就是另一回事——高频电流要走这根线，得克服很大的感抗。

**寄生电容。** 走线和参考平面之间就是电容。容抗的公式：

```
X_C = 1/(2πfC)
```

频率 f 翻 10 倍，容抗降到 1/10。高频信号更容易通过这个寄生电容"漏"到参考平面上去。你要信号往前走，它偏给你跑偏。

**趋肤效应。** 直流电流走导体的整个截面，但交流电流会被"挤"到导体表面。频率越高，电流走的那层"皮"越薄。等效截面积变小 → 电阻变大 → 损耗增加。不是导线变坏了，是物理规律。

**介质损耗。** FR4 板基本身在交变电场中会消耗能量，频率越高越明显。GHz 级别的高频分量在 FR4 里走不了多远就衰减得差不多了。

### 一句话总结

同一根走线，对低频几乎无损，对高频衰减严重。信号频率包含的高频分量越多，走线对它就越不友好。So，一个信号里到底有多少高频分量？这是这一整章要回答的问题。

## 2.2 傅里叶在说什么

傅里叶的核心结论：**任何一个周期信号，都可以分解为若干个正弦波之和。**

### 先看最简单的正弦波

正弦波只有一个频率成分。你在示波器上看它，就是干干净净的一条正弦曲线。

### 再看方波

一个理想方波（上升沿垂直向下，Tr = 0）可以写成一堆正弦波的叠加：

```
方波 =  基频正弦
      + 3次谐波正弦 × (1/3)
      + 5次谐波正弦 × (1/5)
      + 7次谐波正弦 × (1/7)
      + 9次谐波正弦 × (1/9)
      + ...
```

几点关键信息：

**只有奇次谐波。** 2、4、6 次偶次谐波的幅度是零。

**谐波频率：** 基频是 f₀ = 1/周期。3 次谐波就是 3f₀，5 次是 5f₀……

**谐波幅度：** 按 1/n 衰减。3 次谐波的幅度只有基频的 1/3，5 次只有 1/5。越高的频率分量越弱，但永远不为零——理想方波包含的频率是从基频到无穷大。

如果你只取基频 + 3 次谐波，画出来的波形已经像个方波了。加上 5 次、7 次、9 次……谐波越多越接近完美方波。这就是**谐波叠加还原波形的原理**。

## 2.3 频谱是什么

"频谱"就是把波形从**时域**搬到**频域**来看。

- **时域**：横轴是时间，你看到电压随时间变化——这是示波器上看到的。
- **频域**：横轴是频率，你看到每个频率分量的大小——这是频谱分析仪干的事。

方波的频谱：在基频处有一个高的柱子，3 次谐波处有一个矮一点的柱子，5 次谐波处更矮……一直延伸到无穷远。

### 对数坐标

工程上画频谱通常用对数坐标——横轴和纵轴都取对数。为什么？因为频率范围太宽了（Hz 到 GHz 跨 9 个数量级），幅度范围也太宽了。线性坐标根本画不下。

在对数坐标上，理想方波的高频段是一条斜线：每增加十倍频程（频率 × 10），幅度下降 20dB（幅度 × 0.1）。

**-20dB/十倍频程** 这个斜率很关键，后面马上用到。

---

## 2.4 理想方波 vs 真实方波的区别

现实中没有 Tr=0 的方波。真实信号是**梯形波**——上升和下降需要时间。

梯形波在频谱上和理想方波有关键区别。

### 理想方波的频谱

高频段以 -20dB/十倍频程 的斜率衰减，一直到无穷大。也就是说不管多高频，总有一点能量。

### 梯形波的频谱：两个拐点

梯形波的频谱有两个关键的频率拐点：

**第一拐点 f₁ = 1/(π·tₚ)，tₚ 是脉冲宽度。**

频率超过 f₁ 之后，幅度开始以 -20dB/十倍频程下降。这个拐点和方波一样。

**第二拐点 f₂ = 1/(π·Tr)。**

频率超过 f₂ 之后，幅度以 **-40dB/十倍频程** 加速下降。

-20dB/十倍频程 = 频率 ×10，幅度 ×0.1  
-40dB/十倍频程 = 频率 ×10，幅度 ×0.01

衰减速度快了 10 倍。超过 f₂ 后，高频分量被迅速"削掉"，幅度变得极微小。

### Tr 怎么决定一切

Tr 出现在第二个拐点的分母上。Tr 越小，f₂ 越大，加速衰减开始得越晚。

用两个具体数字感受：

| | Tr = 1ns | Tr = 0.1ns |
|:--|:--|:--|
| f₂ = 1/(π·Tr) | ≈ 318MHz | ≈ 3.18GHz |
| 318MHz 以内 | -20dB/dec 温和衰减 | -20dB/dec 温和衰减 |
| 318MHz ~ 3.18GHz | -40dB/dec **加速衰减** | -20dB/dec **仍在温和衰减** |
| 3.18GHz 以上 | — | -40dB/dec 加速衰减 |

关键对比：1GHz 处。

- Tr=1ns 的信号在 1GHz 时已经进入加速衰减段，幅度被压得很低。
- Tr=0.1ns 的信号在 1GHz 时还在温和衰减段，幅度还很可观。

**Tr 减小 → 第二个拐点 f₂ 推高 → 更宽的频率范围内高频分量保留得更多 → 高频能量更足 → SI 问题更严重。**

这就是第一章"Tr 减小是 SI 根源"的数学证明。

## 2.5 信号带宽：哪些频率可以不管

理想方波要完美还原，需要无穷大的带宽——不现实。

工程上定义：**高频分量的幅度已经小到对波形几乎没有影响时，就可以忽略。这个截止频率就是信号带宽。**

### BW = k/Tr

经过大量实验和仿真，总结出了一个简单又好用的经验公式：

```
BW = k / Tr

k = 0.35  →  仪器/放大器的 -3dB 带宽（近似单极点响应）
k = 0.5   →  高速数字 SI 设计时用（多留了裕度）
```

用几个实际芯片跑一下这个公式：

| 信号 | Tr | BW (0.5/Tr) | 意味着什么 |
|:--|:--|:--|:--|
| 74HC 老式逻辑 | 10ns | 50MHz | 基本不用管 SI，连上就行 |
| 74LVC 高速逻辑 | 1ns | 500MHz | 走线超 3cm 就得端接，该控阻抗 |
| DDR4 数据线 | 0.08ns | 6.25GHz | 全程控阻抗、严格等长、低损耗板材 |
| PCIe 4.0 | 0.03ns | ~17GHz | 板材必须是高速料，走线长度严格预算 |

面试被问"信号带宽怎么算"——回答 0.35/Tr 没错。再加一句"做 SI 设计我按 0.5/Tr 留裕度"，面试官就知道你是干过活的。

## 2.6 实践：拿到信号三步判断

实际工作中不需要背那些公式证明，掌握这三步就够了：

![](https://raw.githubusercontent.com/tt1431/hardware-study-plan/main/assets/02-practical-workflow.png)

### 步骤 1：用示波器量 Tr

找到信号的上升时间。注意用合适的探头——被动探头带宽不够会把 Tr 测大，导致你错误判断"这个信号不危险"。

### 步骤 2：算带宽

BW = 0.5/Tr。心里过一下就行。Tr=1ns → 500MHz；Tr=0.3ns → 1.7GHz。

### 步骤 3：判断要不要管 SI

经验阈值：
- BW < 100MHz → 基本不用管，走线就是导线
- 100MHz < BW < 500MHz → 看走线长度。走线延时 > Tr/6 就得端接。FR4 上信号速度约 15cm/ns，Tr=1ns 时临界长度 ≈ 2.5cm。
- BW > 500MHz → 必须认真对待，该控阻抗控阻抗，该端接端接，该等长等长

### 反过来用：选仪器

想量 Tr=0.5ns 的时钟信号：

- 示波器带宽至少 0.5/0.5n = 1GHz，买 1.5GHz 留裕度
- 探头带宽也要 ≥ 1GHz（被动探头通常只有 200-500MHz，得用有源探头）
- 采样率至少要满足 5×BW（Nyquist 大法好，留够裕度）

## 本章小结

这一章的核心就一条线：

```
信号的 Tr → 带宽 BW = k/Tr → 高频分量多少 → SI 问题严重程度
```

- 傅里叶分解告诉你方波 = 一堆正弦波叠加
- 梯形波有两个拐点，Tr 决定第二个拐点在哪
- BW = 0.5/Tr 是做 SI 设计最常用的公式
- 拿到信号量 Tr → 算 BW → 判断需不需要控阻抗
