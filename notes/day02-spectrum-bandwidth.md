## 2.1 一个前提：互连通道对不同频率响应不同

先说一个你每天都在用但可能没明确意识到的事实：同一根 PCB 走线，对 10MHz 和 1GHz 的态度完全不同。

- **寄生电感** — 感抗 ωL，频率翻 10 倍阻抗翻 10 倍。高频电流更难走。
- **寄生电容** — 容抗 1/ωC，频率翻 10 倍阻抗降到 1/10。高频信号更容易漏到参考平面。
- **趋肤效应** — 高频电流只走导体表面，有效截面积变小，电阻变大。
- **介质损耗** — 板材本身随频率升高损耗增大。

所以一根线在低频段几乎无损耗，到了 GHz 级别衰减严重。不是线坏了，是物理规律。这一整章要回答的就是：信号里到底有多少高频成分？这决定了 SI 问题的严重程度。

---

## 2.2 傅里叶干了什么——把方波拆成正弦波

任何一个周期信号都可以分解为若干个正弦波之和。一个理想方波：

```
方波 = 基频正弦 + 3次谐波/3 + 5次谐波/5 + 7次谐波/7 + ...
```

注意几个事实：偶次谐波幅度全部为零；谐波次数越高，幅度越小（按 1/n 递减）。

如果画在对数坐标上，高频分量的幅度每增加十倍频程衰减 20dB——也就是频率翻 10 倍、幅度降到 1/10。

![](https://raw.githubusercontent.com/tt1431/hardware-study-plan/main/assets/02-square-wave-decomposition.png)

打个比方：你看到一个人站在面前，这是"方波"。用傅里叶这把 X 光看，这个人被拆成了骨骼、肌肉、血管——每一层是不同的频率成分。频谱就是把每一层的厚度（幅度）按频率画出来。

## 2.3 真实信号不是理想方波

现实中没有 Tr=0 的方波。真实的数字信号是梯形波。

梯形波和理想方波最大的区别是频谱上多了一个**加速衰减段**：

- 理想方波：高频分量以 -20dB/十倍频一直衰减下去。
- 梯形波：存在两个频率拐点。

第一个拐点 f₁ = 1/(π · 脉宽)，此后开始 -20dB/十倍频衰减。

第二个拐点 **f₂ = 1/(π · Tr)**，此后加速到 -40dB/十倍频衰减。-40dB/十倍频意味着频率翻 10 倍、幅度直接降到 1/100——非常快。

**这个 f₂ 就是 Tr 决定的核心拐点。**

举个例子：

- Tr = 1ns → f₂ = 1/(π·1ns) ≈ **318MHz**。超过 318MHz 后高频分量被迅速削掉。
- Tr = 0.1ns → f₂ = 1/(π·0.1ns) ≈ **3.18GHz**。拐点推到了 3GHz 级别，到 GHz 级别高频能量还很足。

**Tr 越小 → 拐点频率越高 → 信号保留的高频分量越多 → SI 问题越严重。** 这就是"Tr 减小是 SI 根源"的数学证明。

## 2.4 信号带宽：哪些频率可以被忽略

理想方波需要无穷大的带宽才能完美还原——不现实。工程上定一个截止点：高频分量小到对波形几乎没有贡献时，直接忽略。这个截止频率就是**信号带宽**。

## 2.5 ⭐ 核心公式：BW = k / Tr

```
BW = k / Tr

k = 0.35  →  看放大器/仪器规格时用（-3dB 带宽）
k = 0.5   →  做高速数字 SI 设计时用（留了裕度）
```

面试如果被问"信号带宽是多少"，说 0.35/Tr 没问题。加一句"实际 SI 设计我按 0.5/Tr 留裕度"，面试官就知道你是干活的人。

| 信号 | Tr | BW (0.5/Tr) | 设计约束 |
|:--|:--|:--|:--|
| 74HC 老式逻辑 | 10ns | 50MHz | 基本不用管 SI |
| 74LVC 高速逻辑 | 1ns | 500MHz | 走线超 3cm 就得端接 |
| DDR4 数据线 | 0.1ns | 5GHz | 全程控阻抗、严格等长 |
| PCIe 4.0 | 0.03ns | ~17GHz | 高级板材、损耗预算 |

## 2.6 实践中怎么用

拿到一个信号，三步判断它的"危险性"：

![](https://raw.githubusercontent.com/tt1431/hardware-study-plan/main/assets/02-practical-workflow.png)

**反过来用也成立：**

选示波器看 Tr=0.8ns 的信号 → 需要示波器带宽至少 0.5/0.8n = 625MHz，实际买 1GHz 留裕度。

选探头也一样——被动探头带宽通常 200-500MHz，看快于 1ns 的信号得上有源探头。探头带宽不够，量出来的 Tr 偏大，判断就全错了。

---

## 本章小结

> 数字信号的 Tr 决定带宽，带宽决定高频分量多少，高频分量多少决定 SI 问题的严重程度。BW = 0.5/Tr 是你每天都要用的公式。

三条做到就能覆盖 90% 的判断场景：

1. 量 Tr → 算 BW
2. BW > 100MHz 且走线超过 2~3cm → 必须当传输线处理
3. 选仪器/探头时反过来用
