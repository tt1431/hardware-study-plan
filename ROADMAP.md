## 📅 逐日详图

### 阶段一：信号完整性 ⚡ (Day 1-14)

```
Day  1 ── SI概述 ─────────────────── 数字vs模拟 / SI定义 / 根源
Day  2 ── 频谱与带宽 ────────────── 方波频谱 / 带宽公式 / Tr→BW
Day  3 ── 传输线(1) ─────────────── 动态传输 / 返回电流 / 介电常数
Day  4 ── 传输线(2) ─────────────── 特性阻抗 / Z₀影响因素 / 损耗
─────── 🧪 第1周末：阶跃信号反射仿真 ───────
Day  5 ── 反射基础 ──────────────── Γ公式 / 反弹图 / 末端vs入射端
Day  6 ── 容性/感性负载 ─────────── 回勾 / 上冲 / Stub
Day  7 ── 端接(1) ──────────────── 串联/并联/戴维南/AC
Day  8 ── 端接(2) ──────────────── 拓扑结构 / TDR
─────── 🧪 第2周末：端接波形对比仿真 ───────
Day  9 ── 串扰基础 ──────────────── 近端/远端 / 饱和效应
Day 10 ── 串扰深入 ──────────────── 影响因素 / 保护地线 / 9种减串扰法
Day 11 ── 阻抗不连续 ────────────── 分支 / 跨分割 / 过孔
─────── 🧪 第3周末：串扰间距仿真 ─────────
Day 12 ── 差分信号 ──────────────── 差分对 / 共模抑制
Day 13 ── 电源完整性 ────────────── PDN / 噪声来源 / 去耦
Day 14 ── 复习 + 仿真入门 ───────── 知识串联 / 眼图 / TDR
```

### 阶段二：硬件接口 🔌 (Day 15-21)

```
Day 15 ── UART / I2C ────────────── 三线制 / 开漏 / ACK / 上拉阻值
Day 16 ── SPI / CAN ─────────────── 四线制 / 时钟极性 / 120Ω匹配
Day 17 ── USB 2.0 / 3.0 ─────────── NRZI / 位填充 / AC耦合
Day 18 ── Type-C / PD ───────────── CC检测 / Rp/Rd / 正反插 / eMarker
Day 19 ── 以太网 / RGMII ────────── MII→RMII→GMII→RGMII演进 / MDIO
Day 20 ── PCIe ──────────────────── 树形拓扑 / 参考时钟 / SSC / SerDes
Day 21 ── MIPI / HDMI / DDR ─────── D-PHY / TMDS / 握手 / 8位预取
```

### 阶段三：元器件 + 电源 🔋 (Day 22-28)

```
Day 22 ── R/L/C 选型 ────────────── 封装 / 频率阻抗 / X5R vs NP0
Day 23 ── 二极管 / 三极管 / MOS ─── 选型要点 / 电流vs电压控制
Day 24 ── MOS 深入 ──────────────── 米勒平台 / 电平转换 / 驱动电路
Day 25 ── LDO ──────────────────── 拓扑 / 效率 / 压差 / PSRR / 选型
Day 26 ── BUCK(1) ───────────────── 伏秒平衡 / CCM-DCM / PWM-PFM
Day 27 ── BUCK(2) + LDO vs BUCK ── 电感选型 / 自举 / 同步异步 / 损耗
Day 28 ── PCB / 示波器 / 总结 ───── 3W20H / 纹波测试 / 28天汇总博客
```
