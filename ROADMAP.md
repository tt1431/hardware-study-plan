## 📅 逐日详图

### 阶段一：信号完整性 ⚡ (Day 1-7)

```
Day 1 ── SI概述 + 频谱带宽 ──────── 数字vs模拟 / SI根源 / BW=0.35/Tr
Day 2 ── 传输线（完整版）─────────── 特性阻抗 Z₀=√(L/C) / 影响因素
Day 3 ── 反射（完整版）───────────── Γ公式 / 反弹图 / 容感负载 / Stub
Day 4 ── 端接 + 拓扑 ───────────── 串联/并联/戴维南 / 菊花链/TDR
Day 5 ── 串扰（完整版）───────────── 近端/远端 / 保护地线 / 9种方法
─────── 🧪 第1周末：端接波形对比仿真 ───────
Day 6 ── 阻抗不连续 + 差分信号 ──── 分支/跨分割/过孔 / 差分对
Day 7 ── 电源完整性 + 复习 + 仿真 ── PDN / 脑图串联 / 反射仿真
─────── 🧪 第2周末：串扰仿真 ──────────────
```

### 阶段二：硬件接口 🔌 (Day 8-14)

```
Day 8 ── UART / I2C ────────────── 三线制 / 开漏 / ACK / 上拉阻值
Day 9 ── SPI / CAN ─────────────── 四线制 / 时钟极性 / 120Ω匹配
Day 10 ── USB 2.0 / 3.0 ─────────── NRZI / 位填充 / AC耦合
Day 11 ── Type-C / PD ───────────── CC检测 / Rp/Rd / 正反插 / eMarker
Day 12 ── 以太网 / RGMII ────────── MII→RMII→GMII→RGMII / MDIO
Day 13 ── PCIe ──────────────────── 树形拓扑 / 参考时钟 / SSC / SerDes
Day 14 ── MIPI / HDMI / DDR ─────── D-PHY / TMDS / 握手 / 8位预取
─────── 🧪 第2周末：差分信号仿真 ───────────
```

### 阶段三：元器件 + 电源 🔋 (Day 15-21)

```
Day 15 ── R/L/C 选型 ────────────── 封装 / 频率阻抗 / X5R vs NP0
Day 16 ── 二极管 / 三极管 / MOS ─── 选型要点 / 电流vs电压控制
Day 17 ── MOS 深入 ──────────────── 米勒平台 / 电平转换 / 驱动电路
Day 18 ── LDO ──────────────────── 拓扑 / 效率 / 压差 / PSRR / 选型
Day 19 ── BUCK(1) ───────────────── 伏秒平衡 / CCM-DCM / PWM-PFM
Day 20 ── BUCK(2) + LDO vs BUCK ── 电感选型 / 自举 / 同步异步 / 损耗
Day 21 ── PCB / 示波器 / 总结 ───── 3W20H / 纹波测试 / 21天汇总博客
─────── 🧪 第3周末：BUCK + 整体回顾 ───────
```
