# 阶段四：仿真实践指南（贯穿全程）

> 🎯 **目标**：把每天学的理论，用仿真工具亲手验证
> 🔧 **工具推荐**：
> - **KiCad**（免费，自带 SI 仿真基础能力）
> - **LTspice**（免费，电源仿真首选）
> - **HyperLynx / ADS / SIwave**（公司如果有更专业）

---

## 🧪 仿真入门三步走

### Step 1：前仿真（Pre-layout 拓扑分析）

> ⏰ 阶段一后期开始（第 10 天后）

**做什么：**
```
用理想传输线模型搭建电路
  ├─ 改变端接电阻看反射变化
  ├─ 改变线长看时序变化
  ├─ 改变 Tr 看信号质量变化
  └─ 对比不同拓扑的波形差异
```

**KiCad 怎么做：**
KiCad 8.x 内置了 SPICE 仿真器（ngspice），可以搭传输线模型（T-line）做 SI 分析。

**LTspice 怎么做：**
```
元件库有 Lossy Transmission Line / T-line 元件
  └─ Step 信号 + T-line + 负载 → 看反射波形
```

### Step 2：后仿真（Post-layout 验证）

> ⏰ 学会 KiCad 原理图 + Layout 之后

**做什么：**
```
画一块简单的 PCB
  ├─ 一条微带走线 50Ω
  ├─ 一条差分对 90Ω/100Ω
  ├─ 加一个 BUCK 电路
  └─ 提取寄生参数分析
```

**KiCad 怎么做：**
- PCB Editor → 画差分布线
- 用 FreeRouting 或手动布线
- 导出 Gerber 后用 Polar SI9000 算阻抗（有免费版）

### Step 3：眼图分析 + TDR

> ⏰ 进阶玩家

**做什么：**
```
用 PRBS 伪随机码激励
  ├─ 看眼图张开度
  ├─ 看抖动分布
  └─ TDR 测量阻抗不连续点
```

## 📋 每周仿真计划

| 周次 | 对应阶段 | 仿真实验 | 工具 |
|:----:|:--------:|---------|:----:|
| 第 1 周 | SI | 阶跃信号过不同端接的反射波形 | LTspice / KiCad |

## 🔬 重点仿真验证清单

### 信号完整性
- [ ] 反射系数 Γ 公式验证（Z1=50Ω, Z2=75Ω 看反射幅度）
- [ ] 不同 Tr 对反射噪声的影响（Tr 越小反射越大）
- [ ] 串联端接 50Ω 的半幅传输 + 末端反射满幅
- [ ] 容性负载的回勾现象
- [ ] 感性负载的上冲现象
- [ ] 差分信号共模抑制效果

### 电源完整性
- [ ] BUCK 占空比 = Vout/Vin 验证
- [ ] 电感电流在 CCM/DCM 的波形差异
- [ ] LDO 压差对效率的影响
- [ ] 去耦电容 SRF 附近的阻抗特性
