## Day 1 ── SI 概述 + 频谱带宽

**对应章节：** 第一章 + 第二章

| 要点 | 内容 |
|:--|:--|
| 数字 vs 模拟 | 数字信号用模拟波形传输，频率上去所有模拟效应暴露 |
| SI 一句话 | 保证 0/1 码流准确传到接收端并被准确判读 |
| SI 根源 | **$T_r$ 减小是万恶之源**——上升时间越小，高频分量越多 |
| 知识网 | 单章看懂 ≠ 能综合运用，优化一个可能破坏另一个（1.4 小节重点） |
| 电平 vs 边沿 | 电平有效看稳态值，边沿有效看跳变瞬间（时序敏感度差很大） |
| 频谱基础 | 方波 = 无穷奇次谐波叠加；梯形波有两个拐点 |
| 带宽公式 | $BW = 0.35/T_r$（仪器），$BW = 0.5/T_r$（SI 设计留裕度） |
| 核心认知 | 上升时间越长，高频分量衰减越快——所以低速信号 SI 问题不严重 |

**可跳过：** 无
**笔记状态：** ✅ Day1 + Day2 已覆盖

## Day 2 ── 传输线（完整版）

**对应章节：** 第三章

| 要点 | 内容 |
|:--|:--|
| 动态传输 | 信号从源到负载需要时间——必须用"动态"思维 |
| 返回电流 | 瞬时存在，与信号路径同等重要 |
| 参考路径 | 不一定是 GND——VCC/孤立导体也行，电磁波不管网络属性 |
| 介电常数 | 表征和空气相比电容改变多少 |
| 回路电感 | 回路越短电感越小——返回路径靠近信号路径 |
| 瞬时阻抗 | 信号每一步都遇到阻抗，均匀走线 = 各处阻抗相同 |
| **特性阻抗** | $Z_0 = \sqrt{L/C}$，由截面几何决定，与长度无关 |
| 影响因素 | 线宽↓ → $Z_0$↑；介质厚度↑ → $Z_0$↑；$\epsilon_r$↑ → $Z_0$↓ |
| 控制手段 | 主要控线宽和介质厚度（换板材和铜厚成本高） |
| 参考平面作用 | 提供返回路径 + 控制阻抗 |
| 返回电流分布 | 高速信号走阻抗最小路径 = 回路电感最小 = 正下方 |
| 延迟 | $t_{pd} \approx 5.5 \sim 6.5\text{ ps/mm}$（FR4 微带线） |
| 损耗来源 | 阻性损耗（加宽线宽）、介质损耗（换板材）、辐射、阻抗不连续 |
| 趋肤/邻近效应 | 了解现象即可，只能加宽走线或换高速板材 |
| Dk / Df | Dk（介电常数）= 越小越好；Df（损耗角正切）= 越小越好 |

**可跳过：** 趋肤效应公式、表面粗糙度细节
**笔记状态：** ✅ Day3 已覆盖

## Day 3 ── 反射（完整版）

**对应章节：** 第四章（前半）

| 要点 | 内容 |
|:--|:--|
| 反射原因 | **阻抗不连续**——只要阻抗突变就反射 |
| **反射系数** | $\Gamma = (Z_L - Z_0)/(Z_L + Z_0)$，**公式要背** |
| 传输系数 | $T = 1 + \Gamma$ |
| 反弹图 | 跟着算一遍，理解往返叠加过程 |
| 反射本质 | 反射的是**波形**不是数值——反射波形是入射波形的幅度缩小副本 |
| 末端无延迟 | 负载端入射和反射无相对延迟；源端有 $2T_d$ 延迟 |
| 容性负载 | 上电瞬间阻抗=0（短路）→ 充电后→∞（开路）→ 回勾波形 |
| 感性负载 | 上电瞬间阻抗=∞（开路）→ 之后→0（短路）→ 上冲波形 |
| **Tr 的影响** | 上升时间越短，反射噪声越大——低速信号影响不大 |
| 残桩 Stub | 信号传到分支末端反射回来，影响收发两端；分支越短越好 |
| 临界长度 | 看下就行，不重要 |

**可跳过：** 4.14 临界长度细节、4.16 小节、4.17 看结论即可（输出阻抗低不用记）
**笔记状态：** ✅ Day3 已覆盖

## Day 4 ── 端接 + 拓扑

**对应章节：** 第四章（后半）§4.15-4.18

| 要点 | 内容 |
|:--|:--|
| **串联端接** | $R_s + Z_{out} = Z_0$；半幅发射→末端全反射叠加→源端吸收；仅适用于点对点 |
| 串联本质 | 驱动器 $Z_{out}$ 并非恒定值——串阻更准确的作用是**控制加载到线上的电压幅度** |
| **并联端接** | $R_t = Z_0$ 到地；彻底但功耗大：$P = V^2/Z_0$ |
| 末端上拉/下拉 | 上拉到 VCC 拉高低电平；下拉到 GND 拉低高电平 |
| **戴维南端接** | $R_1 \parallel R_2 = Z_0$；适用驱动力不足但必须末端端接的场合；缩小信号摆幅 |
| AC 端接 | 了解就好，基本不会用——电容隔直，只衰减高频噪声 |
| 端接位置 | 串联紧靠驱动器（走线尽量短）；并联放接收器之后（桩线尽量短） |
| 拓扑结构 | 大体看下就行——点对点最理想，菊花链最常见，Fly-by 是菊花链改进版 |
| 链式结构要点 | 桩线越短越好；用末端并联端接（不要串联——前几个分支长期处于中值电平） |
| 总线结构 | 双向传输，两端都要端接，通常戴维南 |
| TDR | 知道是什么即可——利用反射测链路阻抗变化 |

**可跳过：** 拓扑细节只需要知道各自适用场景
**笔记状态：** ✅ Day4 已覆盖

## Day 5 ── 串扰（完整版）

**对应章节：** 第五章 §5.10-5.13

| 要点 | 内容 |
|:--|:--|
| 串扰本质 | 线与线之间只要离得近就有串扰——同一平面、相邻层都一样 |
| 近端远端 | **实际工作不区分**，看一遍直接过，不用深究 |
| 边沿耦合 | 同层走线侧边耦合（最常见） |
| 宽边耦合 | 相邻层走线上下重叠——耦合强度远大于边沿耦合，**相邻层不要重叠走线** |
| 影响因素 | 间距↑↓串扰；耦合长度↑↓串扰；介质厚度↑↓串扰；Tr↓↓串扰↑ |
| **拉开间距** | 最有效减小串扰的手段 |
| **9 种方法** | 间距 > 内层布线 > 缩短平行段 > 端接 > 正交 > 增加Tr > 保护地线 > 低ε板材 > 控阻抗 |
| 保护地线 | 低频模拟信号有用；高速数字信号**谨慎使用**——表层要密集过孔、内层几乎没好处 |
| 蛇形走线 | 突起要矮而宽；内部串扰总是加速信号传播 |

**可跳过：** §5.2-5.4（不用看），§5.5-5.7 看结论
**重点：** §5.10-5.13
**笔记状态：** ✅ Day5 已覆盖

## Day 6 ── 阻抗不连续 + 差分信号

**对应章节：** 第七章 + 第八章

| 要点 | 内容 |
|:--|:--|
| **分支结构** | 分支点阻抗下降（两条传输线并联），分支越长阻抗下降越严重；菊花链要减少分支长度 |
| **参考平面掏空** | 反焊盘附近参考平面变窄，阻抗变化；内层影响比表层小 |
| **跨分割** | 返回路径被切断，阻抗增大+串扰增大；Tr 越小影响越大；敏感信号必须保持连续参考平面 |
| **过孔** | 焊盘→容性、残桩→容性、反焊盘大小可调阻抗；去掉非功能焊盘提升阻抗；残桩越短越好 |
| **差分对** | 接收端有共模抑制能力，共模在较大范围变化都能正常接收 |
| 差分返回电流 | 主要在参考平面，**不是另一条走线**——参考平面对差分同样重要 |
| 差分不一定平行 | 两条线在不同层单独扇出也可以，关键是周围环境一致、减少干扰 |
| 差分端接 | 通常接收端跨接 100Ω |

**可跳过：** 差分对奇偶模理论推导
